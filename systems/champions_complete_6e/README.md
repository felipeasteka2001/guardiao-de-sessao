# Champions Complete - Pacote Operacional para o Guardião

Este diretório transforma o conteúdo de Champions Complete em instruções operacionais para um Agent de RPG.

## Objetivo

Permitir que o Guardião:

- crie personagens de acordo com o HERO System 6E;
- calcule custos em Character Points;
- construa poderes por efeito mecânico;
- aplique Advantages, Adders e Limitations;
- conduza combate por Turn, Segment e Phase;
- acompanhe STUN, BODY, END, condições e Recoveries;
- conceda Experience Points;
- mantenha campanha, ficha e checkpoints consistentes.

## Regra de precedência

1. Configuração específica da campanha.
2. Regras operacionais deste diretório.
3. Champions Complete como fonte de conferência.
4. Decisão explícita do GM.

Quando uma regra não estiver resumida aqui, o Agent deve consultar a referência indicada e não inventar uma regra oficial.
