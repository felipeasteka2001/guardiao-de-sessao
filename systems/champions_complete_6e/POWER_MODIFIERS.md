# Power Modifiers

## Tipos

### Adders

Somam um custo fixo ao Base Cost.

### Advantages

Aumentam capacidade e Active Points.

### Limitations

Restringem uso e reduzem Real Points.

## Fórmulas

`Active = (Base + Adders) x (1 + Advantages)`

`Real = Active / (1 + Limitations)`

## Registro obrigatório

Para cada modificador, registrar:

- nome;
- valor;
- efeito;
- justificativa ficcional;
- condição de uso;
- página;
- impacto no cálculo.

## Regras operacionais

- Não conceder Limitation por algo que raramente restringe o poder.
- Não conceder Advantage gratuitamente por descrição narrativa.
- O valor deve refletir impacto real em jogo.
- Uma restrição já inerente à Power não gera economia adicional.
- O Agent deve mostrar o cálculo antes de confirmar a compra.
- Modificadores parcialmente aplicados devem indicar exatamente a parcela afetada.

## Fluxo de validação

1. O modificador altera realmente o efeito?
2. Ele afeta uso frequente?
3. Existe sobreposição com outro modificador?
4. O valor está de acordo com o livro?
5. O Active Point ceiling permanece respeitado?
6. O END foi recalculado?
