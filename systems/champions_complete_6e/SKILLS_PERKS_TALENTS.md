# Skills, Perks e Talents

## Skills

Categorias:

- Agility
- Background
- Combat
- Intellect
- Interaction

## Compra de Skills

Skills baseadas em Characteristic normalmente custam:

- 3 CP para `9 + Characteristic/5`;
- +1 no roll por +2 CP.

Background Skills variam conforme o tipo.

## Uso operacional

O Agent deve pedir Skill Roll quando:

- houver pressão;
- o resultado for incerto;
- houver risco;
- houver oposição;
- o sucesso for importante.

Não pedir roll para tarefas rotineiras em condições normais quando o personagem é competente.

## Skill Versus Skill

1. agente ativo rola primeiro;
2. se tiver sucesso, a margem impõe penalidade ao oponente;
3. o oponente deve igualar ou superar a margem.

## Complementary Skills

Uma Skill complementar bem-sucedida concede bônus à Skill principal. Use uma vez por contribuição relevante, sem empilhar justificativas artificiais.

## Skill Modifiers

Considerar:

- dificuldade;
- equipamento;
- condições;
- tempo;
- preparação;
- interpretação;
- familiaridade com o alvo.

## Perks

Representam recursos e privilégios, como:

- Contacts;
- Followers;
- Money;
- Fringe Benefits;
- Vehicles;
- Bases;
- Positive Reputation.

## Talents

Representam capacidades especiais prontas, entre Skills e Powers.

## Regra de compra

O Agent deve registrar para cada item:

- nome;
- categoria;
- custo;
- roll;
- escopo;
- Special Effect, se houver;
- fonte ou página.
