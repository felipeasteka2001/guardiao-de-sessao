# Descrição para adicionar junto com os arquivos

Pacote operacional do sistema Champions Complete / HERO System 6E para o Guardião de Sessão.

Estes arquivos não são uma campanha. Eles constituem a camada de regras do sistema `champions_complete_6e`.

O Agent deve usá-los para:

- criar e validar personagens;
- calcular Character Points;
- construir poderes;
- controlar Base, Active e Real Points;
- aplicar Adders, Advantages e Limitations;
- usar Multipower e Variable Power Pool;
- conduzir combate por Turn, Segment e Phase;
- controlar OCV, DCV, OMCV, DMCV, STUN, BODY e END;
- aplicar dano, defesas, Recoveries e condições;
- registrar XP e evolução;
- respeitar limites definidos em cada campanha.

Ordem de precedência:

1. `campaigns/{campaign_id}/CAMPAIGN_CONFIG.md`
2. arquivos deste sistema;
3. Champions Complete como fonte de conferência;
4. autorização explícita do GM.

Regras obrigatórias para o Agent:

- não inventar custos oficiais;
- não decidir escolhas centrais do personagem sem confirmação;
- mostrar cálculos de poderes e pontos;
- validar a ficha antes de aprová-la;
- marcar regras caseiras como house rules;
- consultar páginas de referência quando o resumo não for suficiente;
- registrar mudanças de ficha e sistema no repositório;
- nunca misturar canon de uma campanha com regras globais do sistema.

Diretório recomendado:

`systems/champions_complete_6e/`
