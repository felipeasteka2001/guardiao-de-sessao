# Power Frameworks

## Tipos oficiais

- Multipower
- Variable Power Pool

## Multipower

Usar quando o personagem possui várias aplicações relacionadas de uma mesma reserva de poder, mas não pode usar todas em capacidade máxima ao mesmo tempo.

Registrar:

- reserve;
- slots;
- tipo de slot;
- custo de cada slot;
- distribuição ativa;
- regras de troca.

## Variable Power Pool

Usar quando o conceito exige flexibilidade ampla e legítima, como:

- magia versátil;
- gadgeteering;
- controle de energia;
- adaptação cósmica.

Registrar:

- Pool Cost;
- Control Cost;
- classe de efeitos permitidos;
- Skills exigidas;
- tempo para reconfiguração;
- limitações;
- efeitos proibidos;
- aprovação do GM.

## Restrições operacionais

O Agent não deve usar VPP para conceder qualquer solução sem custo ou sem limite.

Toda configuração deve respeitar:

- conceito;
- Active Point ceiling;
- restrições do Pool;
- tempo de mudança;
- requisitos de Skill;
- efeitos proibidos pela campanha.

## Troca de slots

Trocar slot ou alocação é normalmente uma Zero Phase Action, sujeita às regras específicas do Framework.
